# MultiSoul Site

Official website for MultiSoul, the personal AI agent command center.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Vitest for content-level page checks

## Development

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Checks

```bash
pnpm test
pnpm lint
pnpm build
```

## Deploy

The app is static-friendly and ready to import into Vercel from:

```text
https://github.com/multisoul-ai/multisoul-site
```
